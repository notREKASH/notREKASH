## 📊 Real-world case studies

### 🏭 Guillin (Food packaging manufacturing)
**Problem**: Operators calling forklifts 180 times/day → 12h/day lost  
**Solution**: Real-time notification app (React Native + Node.js API)  
**Result**: **78k€/year** saved in coordination time

### 🛒 Leclerc (Retail distribution)
**Problem**: Perishable goods expiry tracking on paper → stock losses  
**Solution**: Mobile app for expiry date tracking  
**Result**: Identified **tens of thousands of euros** in avoidable waste

### 🏛️ Municipality [in progress]
**Problem**: Manual citizen request management → lost requests, delays  
**Solution**: Request tracking & assignment system  
**Result**: Process optimization in progress
